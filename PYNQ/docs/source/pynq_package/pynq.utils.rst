pynq.utils Module
=================

The pynq.utils module contains functions and classes that are used for delivery
and testing of PYNQ and packages that depend on PYNQ. For delivery it includes
the setuptools function for managing the download of bitstreams from link files
at install time and the enumeration and copying of notebooks. For testing it
provides a function for evaluating a notebook and getting the results.
